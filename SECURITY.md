# Security Policy

## Supported Versions

The supported branch for ownCloud is 2.x.x and for Nextcloud 3.x.x.
The branch 3.x.x is not compatible with ownCloud, and Nextcloud builds
from the 2.x.x branch are no longer released by default.

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

Please do not report vulnerabilities using the normal GitHub issues.
Instead, use the "Report a vulnerability" function found from
https://github.com/nc-music/oc-music/security. You will be credited later
if the report is accepted. The report will remain private until a fix
has been released.
