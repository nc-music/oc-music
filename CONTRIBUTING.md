## Submitting issues

### Guidelines
* This project [oc-music](https://github.com/nc-music/oc-music) is in maintenance
* In most cases, you should report issues and contribute pull requests to the still actively developed Nextcloud version at [nc-music](https://github.com/nc-music/music)
* However, issues and PRs can be filed for this project if they are ownCloud-specific or if you have a justifiable reason to backport something from the Nextcloud version
* __SECURITY__: Report any potential security bug to pauli.jarvinen@gmail.com instead of filing an issue in our bug tracker

### Contribute Code and translations
High-quality pull requests are welcome, following the guidelines given above.
